zch1337 8/8/2019 vue

# 1.创建vuepress项目

注意：node.js >= 8.6

创建基本的项目
1.mkdir vuepress-starter && cd vuepress-starter
2.npm init
3.npm install -D vuepress
4.mkdir docs && echo '# Hello VuePress' > docs/README.md
5.package.json添加
    {
      "scripts": {
        "docs:dev": "vuepress dev docs",
        "docs:build": "vuepress build docs"
      }
    }
6.npm run docs:dev

创建自定义主题的项目
这是主题地址：https://github.com/vuepress-reco/vuepress-theme-reco-1.x.git
1.npm install @vuepress-reco/theme-cli -g
2.theme-cli init

# 2.2默认的页面路由

docs目录作为targetDir,所有的“文件的相对路径”都是相对于 docs 目录。

{
  "scripts": {
    "dev": "vuepress dev docs",
    "build": "vuepress build docs"
  }
}

vuepress dev docs 这里就是从docs开始的
文件的相对路径 页面路由地址
/README.md /
/guide/README.md /guide/
/config.md /config.html
